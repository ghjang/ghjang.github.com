## Common

### Program
* **Quit**: Command + Q
* **Force Quit**: Command + Option + ESC

### File
* **New**: Command + N
* **Open**: Command + O
* **Save**: Command + S

### Edit
* **Select All**: Command + A
* **Copy**: Command + C
* **Cut**: Command + X
* **Paste**: Command + V
* **Undo**: Command + Z
* **Redo**: Command + Y

### Search
* **Find**: Command + F
* **Spotlight Search**: Control + Space Bar

## Caret Movement
* **Move to the End of the current line**: Command + Right Arrow(->)
* **Move to the Start of the current line**: Command + Left Arrow(<-)
